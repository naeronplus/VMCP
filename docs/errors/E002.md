# E002 — REIMPORT_FAILED

**Severity:** Medium

`godot --headless --editor --quit` failed during staging or post-commit reimport after retries (10s, 30s backoff; max 2 retries; 300s timeout).

## Operator actions

1. Download `reimport_failure.log` from S3 diagnostics for the job.
2. Confirm Godot version matches project and export templates (see E006).
3. Check for broken `uid://` references or corrupt resources.
4. Post-commit failures trigger automatic rollback from the pre-commit S3 snapshot.
