# E019 — SCRIPT_OVERRIDE_REQUIRES_ADMIN

Merge patch introduces executable script changes (e.g. `script = ExtResource(...)` or a `.gd` attachment). Requires **`admin`** role on `POST /api/v1/merge` (HTTP **403**).

## Scope (M-02)

**E019 is only for the script-override admin gate.** Invalid job status transitions use **E021** (`INVALID_STATUS_TRANSITION`), not E019.

## Operator actions

1. Re-submit the merge as an admin user, or remove script properties from the operator-tier patch.
2. Review threat model in AGENTS.md (structural merge / script attachments).
3. Deep link: `/api/v1/docs/errors/E019`.
